# Ölmezler Araç Servis Merkezi

Kahramanmaraş'ta Honda özel servis ve araç bakım merkezi web sitesi.

## Özellikler

- ✅ Online randevu sistemi
- ✅ Admin paneli
- ✅ WhatsApp entegrasyonu
- ✅ Google Maps entegrasyonu
- ✅ Mobil uyumlu tasarım

## Çalıştırma

```bash
pip install -r requirements.txt
python app.py
```

- Site: http://localhost:5000
- Admin: http://localhost:5000/admin
- Giriş: admin / olmezler123

## Admin Paneli

- Hero badge yazısı değiştirme
- Telefon numarası değiştirme
- WhatsApp numarası değiştirme
- Randevu yönetimi
